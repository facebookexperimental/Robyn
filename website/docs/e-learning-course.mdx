### Please complete the Robyn Blueprint [**E-learning course**](https://www.facebookblueprint.com/student/path/253121-marketing-mix-models?utm_source=website). It consists of 4 lessons which prepare you to:

---
## 1. [Introduction to Robyn](https://www.facebookblueprint.com/student/path/253121/activity/469849#/page/633f6166422f820a22ceada6?utm_source=website)
- Understand what Robyn is and how it works.
- Identify different modeling approaches.
- Explore the benefits of Nevergrad.
---
## 2. [Data transformation and automation](https://www.facebookblueprint.com/student/path/253121/activity/469850#/page/633f62bf422f820a22ceb20b?utm_source=website)
- Identify data anomalies with exploratory data analysis.
- Define the characteristics of time trend.
- Identify mandatory report columns, given a data input table.
- Calculate adstock exposure, given a scenario.
- Calculate the response for an ad channel, given a graph with data points.
---
## 3. [How to use models](https://www.facebookblueprint.com/student/path/253121/activity/469851#/page/633f62e7422f820a22ceb28d?utm_source=website)
- Install Robyn and Nevergrad.
- Distinguish between organic and paid media variables.
- Determine the type and value of adstock range for facebook_l_variable, given a section of code.
- Recognize the charts included in the output generated by the robyn_run() function.
- Calculate the share of spend, ROAS (return on ad spend), mean spend and % decomposition for facebook_l, given a chart.
- Determine a solution if spend and exposure have a poor fit.
- Select a model, given conflicting conclusions.
---
## 4. [Practical implementations of Robyn](https://www.facebookblueprint.com/student/path/253121/activity/469852#/page/633f6301422f820a22ceb359?utm_source=website)
- Identify the benefits of and actions required for calibration.
- Perform a model refresh.
- Determine the best action to take with a budget allocator, given a scenario.
- Determine whether a conclusion is based on past or current marketing mix averages, given a scenario.
